# NextBuy Backend (simple Express + TypeScript)

Quick start:

1. Copy `.env.example` to `.env` and adjust values.
2. Install dependencies:

```bash
cd backend
npm install
```

3. Run in development:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
npm start
```

Endpoints:
- `GET /api/health` — simple health check
